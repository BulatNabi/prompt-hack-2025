# Формат материалов по предмету

## Общий формат

Материалы передаются в виде **списка строк** (`List[str]`). Каждая строка может содержать:
- Текст лекций
- Конспекты
- Учебные материалы
- Определения и понятия
- Примеры и задачи
- Любой другой текстовый контент по предмету

## Примеры использования

### 1. При старте экзамена

```json
{
  "teacher_name": "Иван Петров",
  "subject": "Математика",
  "teacher_description": "Строгий преподаватель",
  "materials": [
    "Производная функции - это предел отношения приращения функции к приращению аргумента...",
    "Интеграл - это обратная операция к дифференцированию...",
    "Теорема Лагранжа: если функция непрерывна на отрезке..."
  ]
}
```

### 2. При подготовке к экзамену

```json
{
  "teacher_name": "Мария Иванова",
  "subject": "Физика",
  "teacher_description": "Добрый преподаватель",
  "materials": [
    "Законы Ньютона: 1) Тело сохраняет состояние покоя...",
    "Энергия - способность тела совершать работу...",
    "Электрический ток - упорядоченное движение заряженных частиц..."
  ]
}
```

### 3. Длинные материалы

Можно передавать материалы любого объема. Система автоматически:
- Сохранит их в базу данных
- Использует для RAG (Retrieval-Augmented Generation)
- Будет применять при генерации вопросов и ответов

```json
{
  "materials": [
    "Полный текст лекции 1...",
    "Полный текст лекции 2...",
    "Конспект по теме 'Дифференциальные уравнения'..."
  ]
}
```

## Рекомендации

1. **Структурируйте материалы по темам** - каждая строка может быть отдельной темой
2. **Используйте понятный язык** - материалы должны быть понятны LLM
3. **Включайте ключевые понятия** - определения, теоремы, формулы
4. **Добавляйте примеры** - это поможет LLM лучше понимать контекст

## Автоматическое сохранение

Все материалы автоматически сохраняются в базу данных по предмету. При следующих запросах они будут использоваться автоматически, даже если не переданы явно.

## Ограничения

- Материалы должны быть текстовыми (не бинарные файлы)
- Рекомендуется не превышать 10,000 символов на одну строку для лучшей обработки
- Общий объем материалов не ограничен
