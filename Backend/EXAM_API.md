# API для системы экзаменов и подготовки

## Описание

Система позволяет:
1. **Проводить голосовые экзамены** с использованием:
   - **OpenAI** - для генерации вопросов и анализа ответов
   - **Yandex SpeechKit** - для синтеза речи (вопросы преподавателя)
   - **Deepgram** - для транскрипции аудио ответов студентов
   - **RAG** - для использования материалов по предмету

2. **Подготавливаться к экзамену** в текстовом формате с:
   - **OpenAI** - для генерации ответов преподавателя
   - **RAG** - для использования материалов по предмету
   - **Проверкой ухода от темы** - автоматическое возвращение к учебе

## Особенности

- **Реалистичная речь преподавателя** с междометиями ("Кхм...", "Ага...", "Хм...")
- **Эмоциональные реакции** на ответы студентов
- **Автоматическая проверка ухода от темы** - система возвращает к учебе
- **RAG для материалов** - использование контекста предмета

## Переменные окружения

Добавьте в `.env`:
```
DEEPGRAM_API_KEY=your_deepgram_api_key
```

Остальные переменные уже должны быть настроены (IAM_TOKEN, FOLDER_ID, OPENAI_TOKEN, S3_* и т.д.)

## Эндпоинты

### 1. Запуск экзамена

**POST** `/exam/start`

**Запрос:**
```json
{
  "teacher_name": "Иван Петров",
  "subject": "Математика",
  "teacher_description": "Строгий преподаватель, требует точных ответов",
  "materials": [
    "Материал 1 по предмету...",
    "Материал 2 по предмету..."
  ]
}
```

**Ответ:**
```json
{
  "question_id": 1,
  "question_text": "Что такое производная?",
  "question_audio_url": "https://s3.../question_1.ogg",
  "question_index": 0,
  "is_follow_up": false
}
```

### 2. Отправка ответа

**POST** `/exam/answer`

**Запрос:**
```json
{
  "exam_session_id": 1,
  "question_id": 1,
  "answer_audio_url": "https://s3.../answer_1.ogg"
}
```

**Ответ:**
```json
{
  "answer_id": 1,
  "is_correct": true,
  "ai_feedback": "Правильный ответ! Вы хорошо понимаете материал.",
  "teacher_mood": "happy",
  "next_question": {
    "question_id": 2,
    "question_text": "Что такое интеграл?",
    "question_audio_url": "https://s3.../question_2.ogg",
    "question_index": 1,
    "is_follow_up": false
  },
  "exam_completed": false
}
```

Если ответ неправильный, может быть задан follow-up вопрос:
```json
{
  "answer_id": 1,
  "is_correct": false,
  "ai_feedback": "Ответ неполный. Нужно уточнить...",
  "teacher_mood": "disappointed",
  "next_question": {
    "question_id": 2,
    "question_text": "Можете уточнить ваш ответ?",
    "question_audio_url": "https://s3.../question_2.ogg",
    "question_index": 1,
    "is_follow_up": true
  },
  "exam_completed": false
}
```

### 3. Статус экзамена

**GET** `/exam/{exam_session_id}/status`

**Ответ:**
```json
{
  "exam_session_id": 1,
  "status": "in_progress",
  "teacher_mood": "happy",
  "current_question_index": 2,
  "questions_count": 3,
  "created_at": "2025-01-27T12:00:00"
}
```

## Процесс работы

1. **Запуск экзамена**: Студент вызывает `/exam/start` с параметрами преподавателя и предмета
2. **Получение вопроса**: Система генерирует вопрос через OpenAI, синтезирует речь через Yandex SpeechKit и возвращает аудио URL
3. **Запись ответа**: Студент записывает аудио ответ и загружает его в S3 (используя существующий `/upload` эндпоинт)
4. **Отправка ответа**: Студент вызывает `/exam/answer` с URL аудио
5. **Обработка**: 
   - Транскрипция через Deepgram
   - Анализ через OpenAI с учетом контекста и материалов
   - Обновление настроения преподавателя
   - Генерация следующего вопроса или завершение экзамена
6. **Повтор**: Шаги 3-5 повторяются до завершения экзамена

## Настроение преподавателя

Настроение преподавателя меняется в зависимости от качества ответов:
- `neutral` - начальное состояние
- `happy` - правильные ответы
- `disappointed` - неправильные или неполные ответы
- `angry` - серьезные ошибки

Настроение влияет на голос и эмоцию при синтезе речи через Yandex SpeechKit.

## Завершение экзамена

Экзамен завершается автоматически, когда:
- Студент ответил на все основные вопросы правильно
- OpenAI определяет, что экзамен можно завершить (`exam_completed: true`)

## RAG (Retrieval-Augmented Generation)

Материалы по предмету используются для:
- Генерации вопросов, релевантных предмету
- Анализа ответов с учетом контекста предмета
- Сохранения материалов в БД для последующего использования

Материалы можно передать при старте экзамена или они будут загружены из БД, если уже были сохранены ранее.

**Формат материалов:** Список строк (`List[str]`). Подробнее см. `MATERIALS_FORMAT.md`

## Подготовка к экзамену (текстовый формат)

### 1. Запуск сессии подготовки

**POST** `/study/start`

**Запрос:**
```json
{
  "teacher_name": "Иван Петров",
  "subject": "Математика",
  "teacher_description": "Добрый преподаватель, объясняет просто",
  "materials": [
    "Производная функции - это...",
    "Интеграл - это..."
  ]
}
```

**Ответ:**
```json
{
  "study_session_id": 1,
  "teacher_response": "Здравствуй! Я Иван Петров, помогу тебе подготовиться к экзамену по Математика. Задавай вопросы, и я объясню материал."
}
```

### 2. Отправка сообщения

**POST** `/study/message`

**Запрос:**
```json
{
  "study_session_id": 1,
  "message": "Что такое производная?"
}
```

**Ответ:**
```json
{
  "study_session_id": 1,
  "teacher_response": "Кхм... Хороший вопрос! Производная функции - это предел отношения приращения функции к приращению аргумента, когда приращение аргумента стремится к нулю. Понимаешь? Это показывает, как быстро меняется функция в данной точке..."
}
```

### 3. Получение истории сообщений

**GET** `/study/{study_session_id}/messages`

**Ответ:**
```json
[
  {
    "study_session_id": 1,
    "message_id": 1,
    "message_text": "Здравствуй! Я Иван Петров...",
    "is_from_student": false,
    "created_at": "2025-01-27T12:00:00"
  },
  {
    "study_session_id": 1,
    "message_id": 2,
    "message_text": "Что такое производная?",
    "is_from_student": true,
    "created_at": "2025-01-27T12:01:00"
  }
]
```

## Проверка ухода от темы

Система автоматически проверяет, не уходит ли студент от темы учебы. Если студент задает вопросы не по предмету, система вежливо, но настойчиво возвращает к теме:

**Пример:**
- Студент: "А что вы думаете о погоде?"
- Преподаватель: "Кхм... Давай вернемся к предмету экзамена. Мы сейчас готовимся по Математике, давай сосредоточимся на этом."

Эта проверка работает как при экзамене, так и при подготовке.
